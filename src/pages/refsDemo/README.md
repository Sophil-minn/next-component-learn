
## index.js测试入口
### refs
- CustomTextInput.js
  >  DOM 元素添加 ref
- AutoFocusTextInput.jsx
  > 为 class 组件添加 Ref 
- TextInputFunc.js
  > 函数内部使用ref
- CallBackRefs.js
  > 使用 ref 回调函数，在实例的属性中存储对 DOM 节点的引用
- TransferRefs.js
  >  在组件间传递回调形式的 refs

### 转发 refs 
 
- ForwardRef.js
> 使用API React.forwardRef 来获取传递给它的 ref，然后转发到它渲染的 DOM
